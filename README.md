# Accesorios de Mujer — Página demo

Abrir `index.html` en tu navegador para ver la página.

Archivos principales:
- `index.html` — estructura y contenido
- `styles.css` — estilos responsivos
- `script.js` — datos de productos, búsqueda, filtros y carrito

Notas:
- Las imágenes son enlaces externos (Unsplash). Si quieres usar imágenes locales, crea una carpeta `images/` y reemplaza las URLs en `script.js`.
- Para probar localmente, simplemente abre `index.html` con el navegador o usa un servidor local (recomendado para evitar restricciones de CORS al cargar recursos):

  Para PowerShell (Windows) si tienes Python instalado:

  ```powershell
  python -m http.server 8000; Start-Process "http://localhost:8000"
  ```

Si quieres que agregue un panel de administración, más productos, o galerías por categoría, dime y lo preparo.
